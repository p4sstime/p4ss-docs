# PASSTIME.TF Documentation

**PASS Fortress** (previously 4v4 PASS Time) is a 4v4 movement game,

This recommended ruleset book is heavily based on the [ETF2L Rules](https://rules.etf2l.org/); we've updated their rules to be more inline with PASS Fortress / 4v4 PASS Time.
Although this ruleset book will look familiar to ETF2L’s and RGL’s Rules, we still recommend looking through all of the rules as some have been changed to be more applicable to PASS Fortress / 4v4 PASS Time.

The rules are primarily for PASS Fortress, some rules will change depending if you are playing in PASS Fortress or TF2’s 4v4 PASS Time, changes in rules will be marked and explained if different in TF2.

> Any tournament organizer is allowed to tweak these rules. We highly recommend communicating any changes to your players if you use this ruleset as a base.
