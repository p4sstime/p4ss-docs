# <b> 8: Maps and Configs </b>

## <b> 8.1: Global Map Pool </b>

As of the 14th of November 2025, we are using the 4th official version of the global map pool.
Tournament Organizers are required to state the specific map version as maps can get updated during a tournament, unless the tournament organizer makes it clear that the newer version must be played.

Maps are always the latest version unless stated otherwise, with the exact version in the list.

The global map pool includes 8 maps, those being:
<a href=https://passfortress.tf/maps/pass_arena2>Arena2</a>, <a href=https://passfortress.tf/maps/pass_boutique>Boutique</a>, Coastal, <a href=https://old.passtime.tf/maps/pass_greenhouse>Greenhouse</a>, <a href=https://passfortress.tf/maps/pass_ruin>Ruin</a>, <a href=https://passfortress.tf/maps/pass_stadium>Stadium</a>, <a href=https://old.passtime.tf/maps/pass_stonework>Stonework</a>, and <a href=https://passfortress.tf/maps/pass_torii>Torii</a>.

Tournaments are not required to use these maps but are advised to.

# <b>8.2: Config</b>

## <b>8.2.1: Class Limits</b>

Class limits are as follows:

<b>3 Soldiers, 1 Demoman, 1 Medic.</b>

Since teams are composed of 4 players, players may switch their team composition within the class limits as they like.

Playing any other classes by exploiting or a config bug is strictly forbidden.

Players switching to Spectator and/or from Spectator to bypass the class limits or change classes more quickly is strictly forbidden.

# <b>8.2.2: Round and Server Time</b>
Round time is set to 10 minutes via the map.
Server time is always set to 0.

If the round time is set to 5 minutes due to the overwritten round timer from 6v6 configs, please input "round_time_override 600", this will set the round timer to 10 minutes instead of 6v6 5cp config's 5.

# <b>8.2.3: Sv_Pure</b>

Unlike in regular TF2, and other competitive settings, sv_pure 0 has not shown to be problematic as information in PASS Fortress is more readily available, unlike in other settings where players could obtain information that they shouldn't have with use of game modifications. 

HOWEVER: We only advise using custom JACK models, JACK throwing crosshairs, and custom animations with sv_pure 0, due to its limitations. 

# <b>8.2.4: Whitelist</b>

Whitelists are used to ensure that no one uses a weapon that is not allowed.

If the whitelist is not working or bugged out, teams are required to only use the weapons that are usually allowed by the whitelist. If a team uses a weapon that is normally not allowed or not usually accessible, the team(s) may receive a penalty or warning based on the severity of the offense and its impact on the competition.

The current list of allowed weapons can be found on https://whitelist.tf/passtimetf or here:

<b>Taunts:</b>
All taunts are allowed.

<b>Soldier:</b>
Default rocket launcher and its reskins, Black Box, Gunboats, Mantreads, Equalizer, Escape plan, Market Garden, Pain Train, Default melee and it's reskins.

<b>Demoman:</b>
Default grenade launcher and its reskins, Ali Baba's Wee Booties & Bootlegger, Iron Bomber, Splendid Screen, Claidheamh Mor, Scotsman's Skullcutter, Ullapool Caber, Pain Train, Default melee and it's reskins.

<b>Medic:</b>
Crusader's Crossbow, Default Medigun and it's reskins. Amputator, Solem Vow, Ubersaw, Vita-saw, Default melee and it's reskins.

<b>Any weapon that is not listed here such as the default sticky bomb launcher, and the default Syringe gun are not allowed.</b>
